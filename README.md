MyShell
=======

A web-based MySQL administration tool for Hackers. Completely controllable by keyboard.

Commands
----

### Selecting a database

    db <database>

### Show table data

    data <table>

### Insert a new dataset

    insert <table>

### Delete a dataset

    delete <table> <primarykey>

### Edit a dataset

    edit <table> <primarykey>

### Filter a resultset, created by data command

    filter <column> <value>

### Show commands

    help

### Execute custom query

    query <query>
